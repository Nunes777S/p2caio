Introdução ao HTML5 e CSS3 para Iniciantes

Publicado em: 15/06/2023 Tempo de leitura: 12 minutos

Neste tutorial você vai aprender:

  • A estrutura básica de um documento HTML5
  • Principais tags HTML e sua semântica
  • Como vincular CSS ao HTML
  • Seletores CSS básicos e propriedades comuns
  • Criar uma página simples com estilo

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Ele define a estrutura e o conteúdo de uma página através de elementos (tags) que envolvem diferentes partes do conteúdo.

Estrutura Básica de um Documento HTML5

Todo documento HTML5 possui uma estrutura mínima necessária:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
</head>
<body>
    <!-- Conteúdo da página vai aqui -->
</body>
</html>

Principais Tags HTML

HTML5 introduziu elementos semânticos que ajudam a descrever melhor o conteúdo:

  • <header> - Cabeçalho da página ou seção
  • <nav> - Navegação principal
  • <main> - Conteúdo principal
  • <section> - Seção temática do documento
  • <article> - Conteúdo independente e auto-contido
  • <footer> - Rodapé da página ou seção

Exemplo de Uso

<header>
    <h1>Meu Site</h1>
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">Sobre</a></li>
        </ul>
    </nav>
</header>

<main>
    <article>
        <h2>Título do Artigo</h2>
        <p>Conteúdo do artigo...</p>
    </article>
</main>

<footer>
    <p>© 2023 Meu Site</p>
</footer>

Introdução ao CSS3

CSS (Cascading Style Sheets) é usado para estilizar elementos HTML. Ele permite controlar layout, cores, fontes e outros aspectos visuais.

Vinculando CSS ao HTML

Existem três formas de adicionar CSS:

1. CSS Externo (recomendado)

<link rel="stylesheet" href="estilos.css">

2. CSS Interno

<style>
    body {
        font-family: Arial, sans-serif;
    }
</style>

3. CSS Inline (evitar quando possível)

<p style="color: red;">Texto vermelho</p>

Seletores CSS Básicos

Seletores permitem direcionar elementos HTML para estilização:

Seletor Exemplo Descrição
Seletor de elemento p Seleciona todos os elementos <p>
Seletor de classe .destaque Seleciona elementos com class="destaque"
Seletor de ID #header Seleciona o elemento com id="header"

Propriedades CSS Comuns

body {
    font-family: 'Arial', sans-serif;  /* Define a fonte */
    line-height: 1.6;                 /* Espaçamento entre linhas */
    color: #333;                      /* Cor do texto */
    margin: 0;                        /* Remove margens padrão */
}

h1, h2, h3 {
    color: #0066cc;                   /* Cor para títulos */
}

.destaque {
    background-color: #ffffcc;        /* Fundo amarelo claro */
    padding: 15px;                    /* Espaçamento interno */
    border-left: 4px solid #ffcc00;   /* Borda lateral */
}
