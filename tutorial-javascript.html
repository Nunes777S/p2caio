JavaScript Básico para Iniciantes

Publicado em: 20/07/2023 Tempo de leitura: 15 minutos

Neste tutorial você vai aprender:

  • O que é JavaScript e como ele funciona no navegador
  • Variáveis e tipos de dados em JavaScript
  • Operadores e estruturas condicionais
  • Funções e escopo
  • Manipulação básica do DOM

Introdução ao JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web. Diferente de HTML (estrutura) e CSS (estilo), JavaScript é uma linguagem de programação completa que pode:

  • Manipular conteúdo HTML dinamicamente
  • Responder a eventos do usuário
  • Comunicar-se com servidores (AJAX)
  • Armazenar dados no navegador
  • E muito mais!

Incluindo JavaScript em uma Página

Assim como o CSS, o JavaScript pode ser incluído de três formas:

1. JavaScript Externo (recomendado)

<script src="script.js"></script>

2. JavaScript Interno

<script>
    // Código JavaScript aqui
    alert('Olá, mundo!');
</script>

3. JavaScript Inline (evitar quando possível)

<button onclick="alert('Clicado!')">Clique</button>

Variáveis e Tipos de Dados

Em JavaScript (ES6+), usamos let e const para declarar variáveis:

// Declaração de variáveis
let nome = 'João';  // String (texto)
const idade = 30;   // Number (números inteiros ou decimais)
let ativo = true;   // Boolean (true/false)

// Objetos
const pessoa = {
    nome: 'Maria',
    idade: 25,
    profissao: 'Desenvolvedora'
};

// Arrays
const frutas = ['maçã', 'banana', 'laranja'];

Tipos de Dados Principais

Tipo Descrição Exemplo
String Sequência de caracteres (texto) 'Olá', "Mundo"
Number Números inteiros ou decimais 42, 3.14
Boolean Valores lógicos true/false true, false
Array Lista ordenada de valores [1, 2, 3]
Object Coleção de pares chave-valor {nome: 'João', idade: 30}

Operadores e Condicionais

JavaScript possui operadores para realizar operações matemáticas, comparações e lógicas:

Operadores Aritméticos

let soma = 10 + 5;      // 15
let subtracao = 10 - 5; // 5
let multiplicacao = 10 * 5; // 50
let divisao = 10 / 5;   // 2
let resto = 10 % 3;     // 1 (resto da divisão)

Operadores de Comparação

let igual = 5 == '5';    // true (igualdade)
let estritamenteIgual = 5 === '5'; // false (igualdade estrita)
let diferente = 5 != '5'; // false
let maiorQue = 10 > 5;    // true
let menorOuIgual = 10 <= 10; // true

Estruturas Condicionais

// if...else
let hora = 14;

if (hora < 12) {
    console.log('Bom dia!');
} else if (hora < 18) {
    console.log('Boa tarde!');
} else {
    console.log('Boa noite!');
}

// Operador ternário
let mensagem = hora < 18 ? 'Dia' : 'Noite';

Funções

Funções são blocos de código reutilizáveis que realizam uma tarefa específica.

Declarando Funções

// Declaração de função
function saudacao(nome) {
    return `Olá, ${nome}!`;
}

// Expressão de função
const soma = function(a, b) {
    return a + b;
};

// Arrow function (ES6+)
const quadrado = x => x * x;

Chamando Funções

console.log(saudacao('Maria')); // "Olá, Maria!"
console.log(soma(2, 3));       // 5
console.log(quadrado(4));      // 16

Manipulação Básica do DOM

O DOM (Document Object Model) é uma representação do documento HTML que o JavaScript pode manipular.

Selecionando Elementos

// Seleciona pelo ID
const titulo = document.getElementById('titulo');

// Seleciona pelo seletor CSS (primeiro elemento)
const paragrafo = document.querySelector('.destaque');

// Seleciona todos os elementos que correspondem
const botoes = document.querySelectorAll('button');

Modificando Elementos

// Alterando conteúdo
titulo.textContent = 'Novo Título';

// Alterando HTML
paragrafo.innerHTML = '<strong>Texto em negrito</strong>';

// Alterando estilos
titulo.style.color = 'blue';
titulo.style.fontSize = '24px';

// Adicionando/removendo classes
paragrafo.classList.add('importante');
paragrafo.classList.remove('destaque');

Eventos

// Adicionando um event listener
const botao = document.querySelector('#meuBotao');

botao.addEventListener('click', function() {
    alert('Botão clicado!');
});
